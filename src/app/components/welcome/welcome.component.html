Budgeting App

Welcome to my budgeting app. This was made as a project for my portfolio to learn Angular and practice with Postgres. Try out the app and send an issue request through github if you see any glaring problems. I would also like feedback on how to improve the visual design and also the structuring of the app's code.

Features

  1. Can create an account to store all of your data
  2. Can create budgeting groups and items to organize your expense
  3. Can create accounts for all of your money sources and transactions to show where the money goes
  4. Has built in calendar to see all transactions for the month and how they contribute to the total
